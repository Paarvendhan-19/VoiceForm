## 🐛 Troubleshooting

### If build_runner fails:
```bash
flutter clean
flutter pub get
flutter pub run build_runner build --delete-conflicting-outputs
```

### If app doesn't run:
```bash
flutter doctor
flutter clean
flutter pub get
flutter run
```

### If you see errors:
- Check Flutter version: `flutter --version`
- Update Flutter: `flutter upgrade`
- Check device: `flutter devices`
